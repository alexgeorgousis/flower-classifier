Flower Classifier¶

In this project, we trained an image classifier to recognize different species of flowers. We used this dataset from Oxford of 102 flower categories. A few examples can be seen below.

The project was broken down into multiple steps:

  • Load the image dataset and create a pipeline.
  • Build and Train an image classifier on this dataset.
  • Use the trained model to perform inference on flower images.

Load the Dataset¶

We used tensorflow_datasets to load the Oxford Flowers 102 dataset. This dataset has 3 splits: 'train', 'test', and 'validation'. We also made sure the training data is normalized and resized to 224x224 pixels as required by the pre-trained networks (used later).

Create Pipeline¶

In [8]:
# Create a pipeline for each set.
image_size = 224
image_shape = (image_size, image_size, 3)
batch_size = 32

def format_image(image, label):
    image = tf.cast(image, tf.float32)
    image = tf.image.resize(image, (image_size, image_size))
    image /= 255
    return image, label

training_batches = training_set.shuffle(train_set_size//4).map(format_image).batch(batch_size).prefetch(1)
validation_batches = validation_set.map(format_image).batch(batch_size).prefetch(1)
testing_batches = test_set.map(format_image).batch(batch_size).prefetch(1)

Build and Train the Classifier¶

We used the MobileNet pre-trained model from TensorFlow Hub to get the image features. We then built and trained a new feed-forward classifier using those features.

What we did in this section:

  • Load the MobileNet pre-trained network from TensorFlow Hub.
  • Define a new, untrained feed-forward network as a classifier.
  • Train the classifier.
  • Plot the loss and accuracy values achieved during training for the training and validation set.
  • Save the trained model as a Keras model.

# Load the MobileNet pre-trained network from TensorFlow Hub
URL = "https://tfhub.dev/google/tf2-preview/mobilenet_v2/feature_vector/4"
feature_extractor = hub.KerasLayer(URL, input_shape=(image_size, image_size, 3))

# Freeze weights and biases so they're not modified during training
feature_extractor.trainable = False

Image Pre-processing¶

The process_image function takes in an image (in the form of a NumPy array) and returns an image in the form of a NumPy array with shape (224, 224, 3).

In [52]:
def process_image(image):
    image = tf.convert_to_tensor(image, tf.float32)
    image = tf.image.resize(image, (image_size, image_size))
    image /= 255
    return image.numpy()
